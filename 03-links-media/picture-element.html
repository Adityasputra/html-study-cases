Penjelasan Picture Element

<picture> element digunakan untuk memberikan berbagai versi gambar berdasarkan kondisi tertentu seperti:

  • Ukuran layar: Menggunakan media query seperti (min-width: 768px)
  • Orientasi: Portrait atau landscape menggunakan (orientation: portrait)
  • Resolusi layar: Untuk layar retina/high-DPI

Browser akan memilih <source> pertama yang sesuai dengan kondisi. Jika tidak ada yang cocok, akan menggunakan <img> fallback.
